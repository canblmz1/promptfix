LINKEDIN POST:

I got tired of writing bad prompts to AI and getting garbage code back.

Not because the AI is dumb. My prompts were trash.

So I built PromptFix — an open-source, local-first tool that rewrites your lazy prompts into structured, production-ready ones.

Here's what makes it different:

✅ 100% LOCAL — No SaaS, no data leaks. Runs on Groq (API) or Ollama (fully offline).
✅ EVALUATION CENTER — 40 test cases with automated scoring, HTML reports, and CI integration. Because "it works on my machine" isn't enough.
✅ BROWSER EXTENSION — Select text anywhere, press a hotkey, get a better prompt instantly.
✅ UNDERSTANDS TURKISH — "kral login bozuldu başka yeri bozma" becomes a proper software engineering prompt.
✅ BATTERIES INCLUDED — 174 tests, MIT license, CLI + API + extension.

Before: "fix the bug pls"
After: "Investigate and fix the authentication token refresh issue with minimal, targeted changes to the auth middleware..."

Built with Python and vanilla JS because I believe tools should be simple, fast, and under your control.

Try it. Break it. Tell me what sucks.

👉 https://github.com/canblmz1/promptfix
